# 为什么这本书影响了我的设计方法

这本书最大的价值不是告诉你“怎么做一个好看的页面”，而是先重置你对设计的判断标准。

## 我提炼出的 3 个原则

- 先删减信息，再谈视觉装饰
- 用留白建立节奏，而不是堆叠元素
- 让视觉层级服务于内容逻辑

## 在博客上的应用

我把首页改成低噪音布局后，用户能更快看到关键信息，阅读路径也更自然。

# Why this book changed my design workflow

Its core value is not visual decoration. It resets how you evaluate design quality in the first place.

## Three principles I keep using

- Reduce information first, style second
- Use whitespace to create rhythm
- Let visual hierarchy follow content logic

## How I apply it to this blog

After simplifying the homepage structure, visitors reach key content faster and navigate more naturally.
